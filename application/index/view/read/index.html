第1章 新婚夜大战

小说： {$book.book_name} 小说作者： {$book.author_name} 更新时间： {$book.update_time} 书评： 1555 字数： {$book.words}

柳湘迷迷糊糊醒来，发现自己侧躺在冷冰冰的地上。 头痛欲裂，浑身没有一点力气，更不能动弹分毫。 汩汩入耳的，是男人和女人的调笑声。 还有一幅惨烈的图景，在她的脑海里浮现。 红色，刺目的红色，喧闹的喜乐，丰神俊朗却又面露鄙夷的年轻男子。 还有一个媚眼如丝的女人，与那男人一边调笑，一边冷眼看着一袭红色的身影，轰然撞向坚硬的红木桌角。 霎时鲜血四溅，整个世界，沉寂了…… 柳湘记得自己跌入了悬崖，为何醒来想到的，竟是这些？ 她记得，相恋多年的男友勾搭上了富家女，为了扫除她这个“障碍”，将她推下了悬崖。 好像，她才看清了他的真面目，都还没来得及恨他，就已经死了。 既然死了，怎么又会来了这里？ 难道是—— 当“穿越”二字在她的思维中划过，柳湘感觉自己像是被一道惊雷劈到了一样。 待记忆慢慢归位，又恢复了些许体力，她睁开眼睛。 入目看到的，是两个人，一男一女，都穿着大红色的古装，像是一对新婚的夫妻。 但那女人却衣衫不整，香肩裸露，莲藕般嫩白的手臂，如同藤蔓一样，缠绕在男人的脖颈上。 这两人一边笑着，一边看她，见她醒了，面上皆露出鄙夷之色。 柳湘也跟着笑了，那笑里，满是嘲讽。 呵，这便是她的新婚之夜。 呵，这便是她的夫君，司空紫阳。 当看到这一对男女的时候，那些涌入她脑海的记忆，让她明白了刚刚发生过什么事，也让她接受了自己已然穿越的事实。 然而穿越这件事，还是没有眼前这番景象带给她的冲击大。 洞房花烛夜，本该红烛照影，你侬我侬，但那新婚的夫君，竟然当着妻子的面，狎妓淫乱，逼她自尽。 渣男，全都是渣男！ 为这等渣男，柳湘着实觉的那女子死的太不值。 好在她自己并不是为了渣男而寻死的，如若不然，只怕她死了都要被自己气活。 既然她现在占了人家的身体，那就势必要让她活的痛快，不能这么窝囊。 不单单是为了那个死去的女子，也是为了她自己，前世没来得及讨要的，这辈子一并要回来。 柳湘迷迷糊糊醒来，发现自己侧躺在冷冰冰的地上。 头痛欲裂，浑身没有一点力气，更不能动弹分毫。 汩汩入耳的，是男人和女人的调笑声。 还有一幅惨烈的图景，在她的脑海里浮现。 红色，刺目的红色，喧闹的喜乐，丰神俊朗却又面露鄙夷的年轻男子。
